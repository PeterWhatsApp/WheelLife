# Wheel of Life

Native SwiftUI iPhone app for tracking life balance across customizable categories.

## Features (v1.1)

- **Interactive wheel** — drag any segment to rate 0–10 with haptics
- **Curved category capsules** — vivid full-spectrum colors
- **Templates** — Full Spectrum (10) or Classic 8
- **Insights** — live Focus / Strength callouts
- **History** — save snapshots and compare trends
- **Share** — export a polished image of your wheel
- **Reminders** — optional weekly / biweekly / monthly reassessment nudges
- **Home Screen widget** — glanceable balance average + mini wheel
- **On-device only** — no account, no network

## Open in Xcode

1. Open `WheelOfLife.xcodeproj`
2. Select an iPhone simulator or device
3. Press **Run** (⌘R)
4. Add the widget: long-press Home Screen → **+** → search **Wheel of Life**

## App Groups

The app and widget share data via:

`group.Crimson.WheelOfLife`

If Xcode asks you to enable the App Group on your Team, accept it once for both the app and widget targets.

## App Store path

1. Apple Developer account ($99/year)
2. Set **Team**, unique **Bundle Identifier**, and app icons
3. **Product → Archive**, then distribute via App Store Connect
